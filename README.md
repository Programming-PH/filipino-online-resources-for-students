#### Suggested 6-12 Month Learning Sequence Per Track

Use this format for each track:
- `Phase 1 (Month 1-2):` Fundamentals
- `Phase 2 (Month 3-5):` Core tools and guided projects
- `Phase 3 (Month 6-8):` Portfolio projects and deployment
- `Phase 4 (Month 9-12):` Interview prep, certifications, and internships/jobs

##### 1) Front-End Developer

- Month 1-2: HTML, CSS, JavaScript basics, Git/GitHub.
- Month 3-5: Responsive design, React, API integration, testing basics.
- Month 6-8: Build and deploy 3 portfolio projects (landing page, dashboard, e-commerce UI).
- Month 9-12: DSA basics, frontend interview prep, apply to internships/junior roles.

##### 2) Back-End Developer

- Month 1-2: One backend language (Node.js/Python/PHP), HTTP, REST basics, SQL basics.
- Month 3-5: Framework (Express/FastAPI/Django/Laravel), authentication, database design.
- Month 6-8: Build 2-3 APIs (auth service, CRUD app, file upload), deploy with Docker.
- Month 9-12: System design basics, testing, security practices, internship/job applications.

##### 3) Full-Stack Developer

- Month 1-2: HTML/CSS/JS + backend basics + Git workflow.
- Month 3-5: React/Vue + Node/FastAPI + PostgreSQL + authentication.
- Month 6-8: Build end-to-end capstone apps and deploy (frontend + backend + DB).
- Month 9-12: Optimize performance, CI/CD basics, prepare portfolio and resume.

##### 4) Network Engineer

- Month 1-2: Networking fundamentals (OSI/TCP-IP, subnetting, DNS, DHCP).
- Month 3-5: Routing/switching labs, firewall basics, Linux networking tools.
- Month 6-8: Cloud networking (AWS/Azure/GCP VPC concepts), monitoring and troubleshooting.
- Month 9-12: Prepare for entry-level certs (CCNA/Network+) and apply for NOC/network roles.

##### 5) System Administrator (SysAdmin)

- Month 1-2: Linux fundamentals, shell commands, user/process/file management.
- Month 3-5: Bash/Python scripting, package/service management, backups.
- Month 6-8: Automation (Ansible), monitoring, logging, hardening and access control.
- Month 9-12: Cloud VM administration, incident response basics, cert/job prep.

##### 6) DevOps Engineer

- Month 1-2: Linux, Git, networking and scripting fundamentals.
- Month 3-5: Docker, CI/CD pipelines (GitHub Actions/GitLab CI), IaC basics.
- Month 6-8: Kubernetes basics, observability (logs/metrics/traces), cloud deployment.
- Month 9-12: Build production-style pipeline project, interview prep, apply for junior DevOps/SRE roles.

##### 7) Cyber Security

- Month 1-2: Security fundamentals, Linux/network basics, threat concepts.
- Month 3-5: Security tools (Wireshark, SIEM basics), vulnerability scanning, web security basics.
- Month 6-8: Blue-team labs, incident handling workflow, secure coding practices.
- Month 9-12: Entry cert prep (Security+/eJPT paths), CTF participation, SOC analyst applications.

##### 8) Data Analyst

- Month 1-2: Excel/Sheets, SQL basics, descriptive statistics.
- Month 3-5: Data cleaning, visualization (Power BI/Tableau), business reporting.
- Month 6-8: End-to-end analytics projects with dashboards and written insights.
- Month 9-12: Case interview prep, portfolio polishing, apply for analyst internships/jobs.

##### 9) AI / Data Scientist

- Month 1-2: Python, NumPy, Pandas, statistics and probability foundations.
- Month 3-5: Machine learning fundamentals (supervised/unsupervised), model evaluation.
- Month 6-8: Deep learning basics, NLP/computer vision starter projects.
- Month 9-12: Capstone project with deployment, write technical case study, apply for junior ML/data roles.

##### 10) Android Developer

- Month 1-2: Kotlin basics, Android Studio, activity lifecycle, layouts.
- Month 3-5: Jetpack components, local storage, REST API integration.
- Month 6-8: Build and publish portfolio apps (notes app, e-commerce, chat prototype).
- Month 9-12: App optimization/testing, play store readiness, junior mobile dev applications.
